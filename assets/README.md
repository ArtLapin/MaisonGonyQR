# assets

이 폴더에 이미지/폰트 등 정적 파일을 넣습니다.

## 권장 파일

| 파일명 | 용도 | 권장 크기 |
| --- | --- | --- |
| `og-card.png` | 카카오톡·인스타·페북 등에서 링크 공유 시 보이는 미리보기 카드 | 1200 × 630 px |
| `logo.png` (선택) | 텍스트 워드마크 대신 로고 이미지를 쓰고 싶을 때 | 가로 폭 600px 내외, 투명 PNG |

> `og-card.png` 가 없으면 링크 공유 시 미리보기 이미지만 안 보일 뿐, 페이지 동작에는 영향이 없습니다.
> 파일을 추가한 뒤 커밋·푸시하면 자동 반영됩니다.
